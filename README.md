# NextLVL Quest AI Backend for Vercel

## Что это
Vercel-версия backend для AI-генерации квестов в `NextLVL Quest`.

## Endpoint
`POST /api/boards/generate`

## Что нужно на Vercel
Environment Variables:
- `OPENAI_API_KEY`
- `OPENAI_MODEL`
- `NEXTLVL_QUESTS_BACKEND_BEARER`

## После деплоя
Получишь URL вида:
`https://YOUR-PROJECT.vercel.app/api/boards/generate`
